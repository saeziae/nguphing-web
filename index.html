如何安裝

安裝教程

先要安裝小狼毫(Weasel)輸入法 下載

  • 在扗開始菜單裏丄或者狀態欄个「ㄓ」圖標个右擊菜單裏丄揀「輸入法設定」
  • 在彈出个窗口裏撳 「獲取更多輸入方案…」
  • 隨後在彈出个界面，輸入 saeziae/rime_nguphing，隨後撳 Enter
  • 關脫「Rime package installer」个窗口，回轉去「輸入法設定」個版面，擔「吳語拼音」打勾勾，隨後撳「中」
  • 看阿要修改配色，乃末撳「中」完成設定
  • 在扗開始菜單裏丄或者狀態欄个「ㄓ」圖標个右擊菜單裏丄撳「重新部署」
  • 撳 Ctrl 対 `，就好看見「吳語拼音」哉

更新

更新方式同安裝方式是一樣个

先要安裝鼠鬚管(Squerrel)輸入法

下載 或者 通過 homebrew
brew install --cask squirrel

打開終端(Termial)

頭一屆使用請先運行迭隻命令來獲取依賴文件：

curl -fsSL https://raw.githubusercontent.com/rime/plum/master/rime-install | bash -s -- :preset

運行下頭个命令來安裝或更新：

curl -fsSL https://raw.githubusercontent.com/rime/plum/master/rime-install | bash -s -- saeziae/rime_nguphing

請在 default.yaml 中增加輸入法；或者假使尔搞弗清爽並且只需要使用吳語，請使用下頭个自動命令： 

curl -fsSL https://raw.githubusercontent.com/rime/plum/master/rime-install | bash -s -- custom:clear_schema_list custom:add:schema=nguphing custom:add:schema=nguphin-yaehwei custom:add:schema=stroke custom:add:schema=luna_pinyin
  • 箇當中 MacOS 作興會彈出叫尔安裝 git，要撳允許。
  • 運行好仔之後在頂欄「ㄓ」圖標个菜單裏撳「重新部署」。
  • 撳 Control 対 `，就好看見「吳語拼音」哉

爲啥要用吳拼

語言个兩大重點，便是口語対著文字。用吳語拼音打字，一面學習正字，擔使用正字个習慣養起來；一面又鍛鍊自家吳語个發音，讓口語対牢文字連起來。

吳⁠拼/標⁠準吳⁠語弗會得用 如⁠何辦

假使尔會得講吳語，吳拼蠻便當个。尔好在該処瞭⁠解音⁠系⁠表，友⁠好眼个教⁠程我⁠伲也準⁠備提供在許。

雖則標⁠準吳⁠語弗是具⁠體戶蕩个方⁠言，但⁠是對太⁠湖⁠片対台⁠州⁠片个使⁠用⁠者來講起手難⁠度弗是忒大，稍⁠稍⁠裏習⁠慣習⁠慣就來三爻。

假使尔曉得吳⁠語協⁠會式拼⁠音，我⁠伲个輸⁠入⁠法也有得一個吳⁠協⁠式版⁠本（nguphin-yaehwei）。

單字吳⁠語如⁠何唸/拼嘸沒記牢个辰光，覅緊，撳記 ` 到國⁠語反⁠查去，漢⁠語拼⁠音打進去後各各字个吳⁠語發⁠音在屏幕丄就出來哉。
